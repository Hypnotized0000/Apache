# Apache

Apache is een website voor een product: een apparaat dat LoRaWAN gebruikt om
betrouwbaar en energiezuinig data te versturen over grote afstand.

## Stack

- React
- Vite
- CSS

## Bestanden

- `index.html` bevat het React mount-point.
- `src/` bevat de React app, componenten, content en styling.
- `Styleguide.png` is de visuele referentie voor kleuren, typografie en layout.
- `public/media/` bevat het kleine logo, het grote hero-logo en de topography-background.
- `public/fonts/` bevat de lokaal gehoste Space Grotesk fontbestanden.

## Development

Installeer dependencies:

```bash
npm install
```

Start de dev server:

```bash
npm run dev
```

Maak een production build:

```bash
npm run build
```

## Styleguide

De website gebruikt de kleuren uit de styleguide:

- Muted olive: `#9CB760`
- Soft linen: `#E0E2DB`
- Graphite: `#333232`

De uitstraling is rustig, vriendelijk en veilig, met subtiele topografische
lijnen, scherpe hoeken en Space Grotesk als hoofdlettertype.
